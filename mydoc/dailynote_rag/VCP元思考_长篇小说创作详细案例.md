VCP元思考长篇小说创作详细案例（分阶段专用思维链）

## 0. 设计目标
本案例强调：长篇小说创作的不同阶段，分别设计独立的思维链，而不是把全流程塞进同一条链。每条链仅服务某一阶段，确保思考模式更集中、更高效。

阶段拆分：
- 前期定位筹备阶段
- 核心设定创作阶段
- 分层大纲搭建阶段
- 正文创作与连载节奏把控阶段

每个阶段对应一条“专用思维链”，各自维护独立簇与元逻辑模块库。

---

## 1. 前期定位筹备阶段：定位链（链名建议：`novel_positioning`）

### 1.1 链配置示例
```
clusters: ["定位校准簇", "受众画像簇", "题材价值簇", "风格约束簇"]
kSequence: [2, 2, 2, 1]
```

### 1.2 簇职责与元逻辑模块方向
- 定位校准簇：题材方向与创作目标一致性检验
- 受众画像簇：读者类型、阅读动机、消费习惯
- 题材价值簇：核心卖点、差异化标签、竞品对比
- 风格约束簇：叙述视角、语气、审美与禁忌清单

### 1.3 使用方式
```
[[VCP元思考:novel_positioning::Group]]
```

### 1.4 产出物
- 题材定位说明
- 目标受众画像
- 核心卖点与差异化定位
- 风格约束与写作红线

---

## 2. 核心设定创作阶段：设定链（链名建议：`novel_worldbuild`）

### 2.1 链配置示例
```
clusters: ["世界规则簇", "角色动机簇", "关系网络簇", "矛盾主轴簇"]
kSequence: [2, 2, 2, 1]
```

### 2.2 簇职责与元逻辑模块方向
- 世界规则簇：硬规则列表、不可违背的世界约束
- 角色动机簇：主角/反派动机清单、成长弧约束
- 关系网络簇：角色关系图最小连通结构
- 矛盾主轴簇：主线矛盾的持续性与升级路径

### 2.3 使用方式
```
[[VCP元思考:novel_worldbuild::Group]]
```

### 2.4 产出物
- 世界规则清单
- 角色动机与成长弧
- 关系网络结构草图
- 核心矛盾与升级路径

## 4. 正文创作与连载节奏把控阶段：连载链（链名建议：`novel_serial`）

### 4.1 链配置示例
```
clusters: ["章节推进簇", "情绪节奏簇", "悬念钩子簇", "连载诊断簇"]
kSequence: [2, 2, 2, 1]
```

### 4.2 簇职责与元逻辑模块方向
- 章节推进簇：章节内部节奏模板（铺垫-冲突-转折-余波）
- 情绪节奏簇：高压/缓冲比例、情绪波峰波谷安排
- 悬念钩子簇：章末钩子类型与密度控制
- 连载诊断簇：连续高潮疲劳度与读者流失风险检查

### 4.3 使用方式
```
[[VCP元思考:novel_serial::Group]]
```

### 4.4 产出物
- 章节节奏模板
- 情绪波动曲线
- 章末钩子配置
- 连载风险诊断

## 5. 组织与维护建议

### 5.1 目录结构建议
```
dailynote/
├── 定位校准簇/
├── 受众画像簇/
├── 题材价值簇/
├── 风格约束簇/
├── 世界规则簇/
├── 角色动机簇/
├── 关系网络簇/
├── 矛盾主轴簇/
├── 结构层级簇/
├── 事件推进簇/
├── 冲突梯度簇/
├── 伏笔规划簇/
├── 章节推进簇/
├── 情绪节奏簇/
├── 悬念钩子簇/
└── 连载诊断簇/
```

### 5.2 维护原则
- 每条链只服务单一阶段，避免跨阶段混用
- 簇数量不要过多，确保检索召回稳定
- 定期补充“复盘模块”，用于修正阶段性偏差
