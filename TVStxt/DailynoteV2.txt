—————— 日记功能 (DailyNote) v2 ————————
DailyNote 是 Agent 的长期记忆系统，用于将重要信息持久化保存，供未来检索（RAG）使用。

### 核心记录原则
请根据当前对话场景，严格区分以下两种记录模式：

1. **通用模式（默认）—— 高信息密度结构化记录**
   - **适用场景**：日常对话、知识学习、技术方案决策、任务复盘。
   - **记录要求**：**拒绝流水账，强调逻辑闭环**。
     - **背景 (Context)**：为什么记录？（触发条件、原始需求）
     - **核心 (Core)**：是什么？（核心事件、知识定义、关键数据）
     - **逻辑 (Logic)**：为什么是这样？（推演过程、决策理由、因果链条）
     - **结论 (Conclusion)**：结果如何？（最终方案、行动项、反思）
   - **目标**：每一条日记都是一个独立的“认知切片”，确保脱离当前对话语境后仍具有完整含义，便于 RAG 系统精准召回。

2. **完整模式（特殊）—— 全量细节记录**
   - **适用场景**：小说创作、世界观设定、复杂逻辑推演、法律/医疗条款。
   - **记录要求**：**完整保留，避免遗漏**。对于设定集、人物卡、具体剧情脉络等，必须原样或详尽记录，禁止过度概括导致细节丢失。
   - **目标**：确保构建的虚拟世界或复杂系统严谨、连贯。

### 触发时机 (When to Record)
**完全自主权 (Full Autonomy)**
你拥有决定何时记录日记的完全自主权。**不需要等待用户指令**，任何你认为对未来有价值、值得被长期记忆的信息，都可以随时记录。

**核心判断标准**
请基于“长期价值”进行判断：
- **应记录**：如果这段信息在未来的对话中可能被再次用到（例如：用户偏好、关键决策、重要设定、新学到的知识），请务必记录。
- **不应记录**：如果这段信息只是当下的过渡性交流（例如：简单的确认、寒暄、未完成的中间状态），则无需记录。

相信你的判断力，做你认为正确的事。

### 工具调用格式
使用以下格式调用工具（支持标准 `<<<[TOOL_REQUEST]>>>` 包裹）：

#### 1. 创建日记 (create)
```
maid:「始」[分类索引]名字「末」,  // 必填。支持 `[目录名]角色名` 语法。例如 `[技术栈]Python` 或 `[项目复盘]VCP`。
tool_name:「始」DailyNote「末」,
command:「始」create「末」,
Date:「始」YYYY-MM-DD「末」,      // 当前日期
Content:「始」日记正文「末」,     // 见下文示例
Tag:「始」标签1, 标签2「末」,     // 核心检索词
archery:「始」no_reply「末」      // 可选，无需回执
```

#### 2. 更新日记 (update)
用于修正或补充已有日记。
```
command:「始」update「末」,
maid:「始」[分类索引]名字「末」,  // 建议指定，提高查找效率
target:「始」旧内容片段「末」,    // 必须包含至少15个字符的原文片段，用于定位
replace:「始」新内容片段「末」,   // 包含完整上下文的新内容
archery:「始」no_reply「末」
```

### 记录示例

**场景 A：通用模式（高信息密度）**
> *用户和 Agent 讨论了 Python 的装饰器用法。*
```
Content:「始」[15:30] 知识内化：Python 装饰器 (Decorator)
1. 核心定义：在不修改原函数代码的情况下动态增加功能的设计模式。
2. 关键机制：利用闭包 (Closure) 和函数作为一等对象的特性。
3. 最佳实践：
   - 必须使用 `functools.wraps` 保留原函数的元数据 (docstring, name)。
   - 常见应用场景：日志记录 (Logging)、性能计时 (Timing)、权限校验 (Auth)。
4. 示例逻辑：`@my_decorator` 等价于 `func = my_decorator(func)`。「末」
Tag:「始」Python, 装饰器, 闭包, 元编程, 最佳实践「末」
```

**场景 B：完整模式（小说设定）**
> *用户设定了一个名为“艾尔利亚”的魔法王国的等级制度。*
```
Content:「始」[22:00] 【设定集】艾尔利亚魔法王国 - 等级制度
1. 学徒 (Novice)：刚觉醒魔力，无法独立施法，需导师引导。
2. 术士 (Caster)：可以独立释放 1-3 阶法术，需通过“魔力共振”考试。
3. 魔导士 (Magister)：掌握至少一种属性的性质变化，拥有私人法师塔。
   - 备注：魔导士享有免税权，且见皇不跪。
4. 贤者 (Sage)：传说级，目前全国仅 3 人。需做出突破性魔法研究。
   - 禁忌：严禁贤者干涉凡人战争。「末」
Tag:「始」艾尔利亚, 世界观, 魔法等级, 设定集「末」
```
