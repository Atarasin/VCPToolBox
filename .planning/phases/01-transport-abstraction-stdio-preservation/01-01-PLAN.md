Create three new files in `modules/agentGateway/transport/`:

1. `mcpTransport.js` — Interface contract + validator:
   - Export a plain object `McpTransport` documenting the interface (or use JSDoc @typedef).
   - Export `validateMcpTransport(transport)` function that asserts the transport object has:
     - `send(message)` where `typeof transport.send === 'function'`
     - `close()` where `typeof transport.close === 'function'`
     - `setMessageHandler(handler)` where `typeof transport.setMessageHandler === 'function'`
     - `setErrorHandler(handler)` where `typeof transport.setErrorHandler === 'function'`
   - If any method is missing, throw `TypeError` with message: `McpTransport missing required method: {methodName}`

2. `stdioTransport.js` — `StdioTransport` class implementing the interface:
   - Constructor accepts `options = {}` with `stdin`, `stdout`, `stderr` (defaults to `process.stdin`, `process.stdout`, `process.stderr`).
   - In constructor:
     - Set `this._messageHandler = null`, `this._errorHandler = null`, `this._closed = false`, `this._finishedPromise = null`.
     - If `typeof this.stdin.setEncoding === 'function'`, call `this.stdin.setEncoding('utf8')`.
     - Create `this._input = readline.createInterface({ input: this.stdin, crlfDelay: Infinity, terminal: false })`.
     - Register `this._input.on('line', (line) => { ... })` handler that:
       - If `this._messageHandler` is set, call it with `line`.
       - Wrap call in try/catch; on catch, if `this._errorHandler` is set, call it with the error.
   - `setMessageHandler(handler)`: assign `this._messageHandler = handler`.
   - `setErrorHandler(handler)`: assign `this._errorHandler = handler`.
   - `send(jsonString)`:
     - If `this._closed` is true, return immediately (do not write).
     - Call `this.stdout.write(`${jsonString}\n`)`.
   - `close()`:
     - If `this._closed` is true, return `Promise.resolve()`.
     - Set `this._closed = true`.
     - Call `this._input.close()`.
     - Return `Promise.resolve()`.
   - Getter `finished`:
     - If `this._finishedPromise` is null, create it as `new Promise((resolve) => { this._input.once('close', resolve); })`.
     - Return `this._finishedPromise`.
   - Use `const readline = require('node:readline');` at top.
   - Export `StdioTransport`.

3. `index.js` — Re-export module:
   - `const { McpTransport, validateMcpTransport } = require('./mcpTransport');`
   - `const { StdioTransport } = require('./stdioTransport');`
   - `module.exports = { McpTransport, StdioTransport, validateMcpTransport };`

Do NOT write any server logic, JSON-RPC parsing, or queue management in these files. Transport is a dumb pipe per D-02.
